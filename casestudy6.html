Non-profit provider database - THE CHILDREN'S CAMPAIGN

The problem

While at the Children's Campaign, I was given the opportunity to work on the creation of their donation-based MS Access database project. Already being built-upon by past database apprentices, my task was to continue to build upon it.

The proposed solution

Due to other prioritized projects, I was unable to complete the database. However, I was able to further edit the front-end menu designs, write staff and training documentation for the database, and gain more experience with MS Access database creation.



Front-end login page


Individual contribution
  • Tested the database to assure that all changes were error-free.
  • Edited the front-end menu design(can be seen in the image above).
  • Wrote documentation on how to utilize(for all staff) and edit(for technology staff)the database.

The Impact

  • Learned more about how to use MS Access.
  • Provided further progress toward the completion of the database.
  • Created guidelines that can be used to help the staff easily adapt to the database.
