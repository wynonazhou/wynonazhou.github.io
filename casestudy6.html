Non-profit provider database - THE CHILDREN'S CAMPAIGN

As an intern at the Children’s Campaign, my main specialty was Database. Because of this, I worked on many tasks involving on creating databases that would be used by those in the organization. Some other databases that I have created throughout my time at the Children’s Campaign include a donation database that allows users to enter in Paypal records and extract them, as well as a database that allows users outside the organization to enter data in their own personal accounts.

For now, I will discuss one of the more completed databases. This database being a provider centered database that is used by those inside the organization with similar criteria used through previous databases such as the donation database. For this database, the organization would like a reservoir where they can enter in records of users and extract them into csv. The database needs to have a log in systems and it must be used by those only in the organization.

The proposed solution

From all the criteria I was given before I started the process, the best route to go for this project was using Microsoft Access. Previous databases used in the non-profit had been created using MS Access, so to reduce confusion for future use or edits, it would be best to keep the process similar. With a few new edits and additions done, the database was finished with great success and it has been used by organization members on a daily basis.
